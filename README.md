# Tracky

**NOTE** Under development. Not ready for production use

Ruby gem for the [Tracky API](http://tracky.com/).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tracky'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tracky
